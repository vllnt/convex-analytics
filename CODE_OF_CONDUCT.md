# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

We are committed to providing a friendly, safe, and welcoming environment for all contributors, regardless of experience level, identity, or background.

Please be respectful, constructive, and professional in all interactions.

## Reporting

Report conduct issues via [Discord](https://bntvllnt.com/discord) or by opening a GitHub issue for non-sensitive matters. For security vulnerabilities, use [GitHub Security Advisories](https://github.com/bntvllnt/convex-analytics/security/advisories).
